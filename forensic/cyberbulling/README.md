# Backdoor - Кибербулинг
**Сложность:** medium
## Описание
Какой-то злой шутник решил произвести кибербулинговую аннигиляцию ментального здоровья нашего системномного администратора. У бедняги уже глаз дёргается из-за кибербулящей папки в корневой директории, которая появляется после каждой перезагрузки. Помоги нам разобраться в природе этого явления, пока админа не увезли в дурку на транкотерапию...<br>

## Настройка среды
1) Распаковать образ, чтобы получить archlinux.qcow2
2) Скачать qemu под архитектуру x86\_64 и virt-manager
3) В virt-manager создать виртуальную машину
4) QEMU/KVM, Import existing disk image
5) Путь до archlinux.qcow2, в поле ниже ввести archlinux
6) Нажимать далее, пока не установится
7) Логин/пароль: root/root
